# polymedia

## Overview
##This is a video sharing, streaming platform
# Backend is built with Go and Gin, Frontend With React, Database used are PostgreSQL and MongoDB and object storage is MiniO
# Notification service with Elixir
# Video processing pipeline with C++ and FFMPEG
# Redis will be used for caching
